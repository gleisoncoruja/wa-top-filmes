Início

                    API Endpoint

                    http://localhost:5000/api

A WA Top Filmes API fornece 50 filmes que estão em alta, a cada pagina a API retorna 10 resultados, com ednpoints simplificados e muito simples de utilizar

Para usar essa API não é necessário uma API Key, bastando apenas acessar as rotas.

URL Base: http://localhost:5000/api/

Listar Filmes


# curl  exemplo
curl \
-X GET http://localhost:5000/api/movies

Para listar os filmes bastar fazer um GET conforme a URL abaixo :
http://localhost:5000/api/movies
